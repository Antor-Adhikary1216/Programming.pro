# Frontend

React frontend powered by Vite with Tailwind CSS, React Router, React Icons, and a fetch-based API layer.

## Scripts

- `npm run dev` — start the development server.
- `npm run build` — create a production build.
- `npm run lint` — run Oxlint.

Copy new environment variable names into `.env.example`. Only variables prefixed with `VITE_` are exposed to frontend code.
